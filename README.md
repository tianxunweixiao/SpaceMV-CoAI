# SpaceMV-CoAI-Wp/Wc
（产业星球运营智能体—内容生产及运维工具）

## 1 SpaceMV-CoAI 的愿景

### (1) SpaceMV-CoAI 的角色：
产业星球首席运营官（Chief Operating Officer ，缩写 COO）。

### (2) SpaceMV-CoAI 的愿景：
再小的个体，也有自己的星球。

### (3) SpaceMV-CoAI 的内核：
**Cooperation（合作）**：像管理初级员工一样管理 AI Agent。

> **备注**：作为一个负责 Vercel 全球业务的 COO，Jeanne 非常关注 AI 提效。她提出了一个非常有趣的观点：管理 AI Agent 并不需要一套全新的技能，它其实就是管理学基本功的回归。
>
> “管理 AI 智能体的核心技能，实际上与管理那些需要明确指令、反馈和纠偏的初级员工完全一致。” —— Jeanne DeWitt Grosser
>
> 她建议团队在引入 AI 工具时，要像给新人做入职培训一样，提供清晰的背景、明确的目标以及容错空间。在 Vercel 内部，他们并不盲目迷信现成的 AI 工具，而是强调自定义和针对企业私有代码/数据的微调，因为“通用的 AI 永远无法解决深层的问题”。

### (4) SpaceMV-CoAI 的目标：
按照 APICS(美国生产与库存控制协会，现在已更名为 ASCM，即 Association for Supply Chain Management）的定义，运营管理是对将输入转化为成品和服务的活动进行计划、调度和控制的过程。

SpaceMV-CoAI 将助力企业战略转换与运营规划、体系构建与效率提升、组织协同与团队赋能等方面实现智能化转型：

*   **一是内容生产及运维工具**：企业官网、自媒体等内容生产、审核、发布、监测（流量监测/防篡改等安全监测）、统计、分析（诊断）、反馈（质量/绩效评价）一站式服务。
*   **二是产业星球**：接入SpaceMV-ScAI（由成都天巡微小卫星科技有限责任公司研发的面向 Agent 架构的开源星座智能管理平台，GitHub地址：https://github.com/tianxunweixiao/SpaceMV-ScAI）对重点产业园区过境预报、覆盖性分析等的一键发布。
*   **三是产业仪表盘**：通常指用于监测和分析特定产业或行业运行状况的可视化工具，包括不限于特定行业仪表盘、经营仪表盘、供应链仪表盘、跨行业关联仪表盘等。通过整合关键数据指标，支持用户自定义图表组合，进行数据对比、相关性分析与历史回测，辅助投资及经营决策。
*   **四是交互式产业地图**：结合地理信息系统（GIS）、时空大数据和交互技术的数字化工具，通过整合实时数据，如企业分布、产业规模、创新资源等，将传统静态地图升级为动态“实景图”，支持用户按产业类别、区域或功能定位多维检索，精准查询产业布局，辅助制定产业政策、优化空间布局，引导资源集聚；覆盖规划、政策、园区、企业、人力资源等多类数据，构建全域产业“一张图”，帮助用户跨区域对比产业结构与资源禀赋，助力企业精准选址、拓展市场，降低投资风险；支持用户自主操作，如缩放、切换图层、关联查询等，集成政策查询、资源对接等功能，提供实时产业信息。

## 2 SpaceMV-CoAI-Wp/Wc 介绍
SpaceMV-CoAI-Wp 和 SpaceMV-CoAI-Wc 是 SpaceMV-CoAI 产品生态中的两个 MCP 应用，它们解决的是企业内容营销活动中面临的多平台内容分发耗时耗力、运营数据分散难以洞察、内容审核流程复杂低效的难题。

### ■ SpaceMV-CoAI-Wp：官网内容管理，灵活掌控品牌门面
SpaceMV-CoAI-Wp 是成都天巡微小卫星科技有限责任公司官网（http://spacemv.com/ ）的内容中枢。它提供了直观、强大的内容编辑功能，同时它提供了基于插件化的灵活页面布局需求，让您能够轻松实现：

*   **版面插件化设计**：每一种官网版面风格都被设计为一个独立的插件（Plugin）。
*   **完全可定制**：SpaceMV-CoAI-Wp 支持用户导入这些插件，并进行页面布局、样式和内容的完全可编辑、可定制。您不再受限于成都天巡微小卫星科技有限责任公司固定的页面风格，可以根据品牌调性自由设计。
*   **丰富的插件库**：SpaceMV-CoAI-Wp 将提供一系列定制好的版面插件供用户选择，实现“开箱即用”与“深度定制”的完美结合。
*   **图文替换**：快速更新产品图片、活动海报和文字信息。
*   **实时发布**：确保官网内容始终保持最新状态，第一时间向用户传递品牌声音。

### ■ SpaceMV-CoAI-Wc：自媒体一站式创作与高效分发
对于营销团队而言，SpaceMV-CoAI-Wc 彻底革新了自媒体运营模式。想象一下，您不再需要频繁登录微信公众号、小红书等多个平台，重复进行图文编辑和排版。

*   **统一创作平台**：创作者只需在 SpaceMV-CoAI-Wc 中完成一次图文编辑，即可适配多个自媒体平台的发布要求。
*   **基于角色的审核机制**：内置严谨的角色权限与审核流程，确保每一篇推文都经过公司领导或指定人员的审核，保障内容质量和品牌安全。
*   **高效运营与时效性**：无论是文章发布还是紧急下架，SpaceMV-CoAI-Wc 都能实现一键同步操作。这极大地降低了多平台运营的时间成本和人力成本，让您的内容运营更具时效性和可控性。

## 3 SpaceMV-CoAI-Wp/Wc 的架构解析
从技术实现上，SpaceMV-CoAI-Wp/Wc 主要有以下四个关注点：

### 架构关注点一：技术选型与云原生形态
*   **核心框架**：SpaceMV-CoAI-Wp/Wc 基于国内成熟稳定的 **若依（RuoYi）框架** 进行开发，确保了系统的快速迭代和高可维护性。
*   **云原生设计**：从技术形态上，我们完全遵循云原生理念。系统采用 Docker 容器化部署，并运行在 **K8S 容器编排框架** 上，这使得 SpaceMV-CoAI-Wp/Wc 具备了服务动态扩容和高可靠运行的能力，轻松应对流量高峰。

### 架构关注点二：创新插件化架构，实现页面动态构建
为实现官网版面的完全可定制化，我们引入了创新的插件化架构：

*   **版面插件管理中心（PluginMgr）**：负责管理用户导入和官方提供的所有版面插件，包括插件的配置、版本控制和权限管理。
*   **动态渲染与加载引擎（DynamicLoader）**：这是实现定制化的核心。它能够根据用户在后台的布局配置，动态加载所需的插件，并结合内容数据进行实时渲染，最终呈现出完全个性化的官网页面。

### 架构关注点三：数据仓库机制，支撑深度分析
为了实现对运营数据的深度挖掘和历史分析，SpaceMV-CoAI-Wp/Wc 创新性地集成了 **数据仓库（Data Warehouse）** 机制：

*   **原生数据抓取**：系统能够抓取各自媒体平台原生数据，确保数据来源的准确性和完整性。
*   **ClickHouse 赋能**：我们选取了高性能的列式存储数据库 **ClickHouse** 作为数仓数据库，它能够支撑不同主题、不同维度的历史分析和实时分析，为您的运营决策提供秒级响应的数据支持。

### 架构关注点四：MCP 赋能 Agent 员工
通过 **MCP（Model Context Protocol）** 封装，可以作为 Tools 集成到 SpaceMV-CoAI 中。以实现：

*   **成都天巡微小卫星科技有限责任公司内部数字运营员工**：通过语义化交互方式，智能体自动化完成自媒体内容运营各项操作
*   **第三方集成**：任何遵循 MCP 协议的第三方智能体平台都可以轻松集成和调用 SpaceMV-CoAI 的内容管理和数据分析能力
